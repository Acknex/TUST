Detailed Description

Extendable and collapsable menu panels manager.

The compact menu module is a tool that lets you create and use collapsable menus very easily. It is also conceived to be easily extensible with your own compact_menu members.

How it does work

Every compact_menu is a secuence of members that are shown, hidden or modified with the mouse interaction. Compact_menu parses the content of strings in order to build the data structs needed to manage the automatic menu drawing and menu member selection.

General member syntax rule

name.member_type=parameters
Parameter Description
name text to be displayed in the left side of the member field.
member_type is the member type name.
parameters one or more parameters in reference to member_type separated by commas.

Default Members

submenu
name.submenu=TEXT*

The most important member in compact menu. It lets us create collapsable menus. Each submenu is created in reference to a TEXT struct that contains all the member construction strings of the submenu.

Parameter Description
TEXT* TEXT struct that contains the strings to be parsed as members of the submenu.

Example:

TEXT *txtMenu =
{
string = (
".line=0",
"camera.submenu=txtCamera",
"sun.submenu=txtSun",
"ambient.submenu=txtAmbient",
".line=2" );
}

digit
name.digit=decimal,var
name.digit=decimal,var,event

Draws the content of a var. If the var is a pointer to a var, the digit will show the content of the var that was pointed by the pointer in the moment of the menu creation. If the a digit var is continued by a function, the content of the var will be set to the return of the function while the digit is visible. Conceived to show menu related temporary values.

Parameter Description
decimal decimal number. 0 <-> 3
var variable to be shown.
event function to fill the var.
digedit
name.digedit=decimal,var
name.digedit=decimal,var,event

Draws and edit the content of a var. If the var is a pointer to a var, the digedit will show the content of the var that was pointed by the pointer in the moment of the menu creation. The digedits are edited by clicking over them and moving the mouse up and down. The change difference depends on the horizontal position of the mouse.

Parameter Description
decimal decimal number. 0 <-> 3
var variable to be shown and edited.
event function to be called after editing the digedit. It can be obviated.
