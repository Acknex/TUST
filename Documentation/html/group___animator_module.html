Starts to blend from one animation to another.

This function starts the blending of two animations. The first animation is the current animation of the controller, the second animation is the animation defined in the parameters.

Parameters
animatorThe animation controller that should blend the animation.
animationThe animation that should be blended to.
ticksDuration of the blending in ticks (1/16 sec).
