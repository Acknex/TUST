Detailed Description

Library for dynamic model creation.

DynamicModels.h allows you to create new models at runtime.

#define D3DXF_FILEFORMAT_TEXT   1

Save dynamic models in a text x format.

#define DMDL_CLONE_TEX   (1<<2)

DynamicModels System Flag. Clones the textures of a newly created entity from the original ones.

#define DMDL_FIXNORMALS   (1<<0)

DynamicModels System Flag. Activates normal fixing when creating the mesh. Normals will be smoothed out and calculated correctly for each vertex.

#define DMDL_MAX_INDEXCOUNT   128000

Maximum index count of a dynamic model.

#define DMDL_MAX_VERTEXCOUNT   65536

Maximum vertex count of a dynamic model.

#define DMDL_OPTIMIZE   (1<<1)

DynamicModels System Flag. Activates mesh optimizations when creating the mesh. Vertices will be welded if they are close together and enables a smoother lighting and faster rendering.
