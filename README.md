TUST
====

This project is a scripting library and game template collection for Gamestudio A8.
We try to provide a starting base for beginners and a script library for extending the
Gamestudio API.

If you want to contribute with models, scripts or templates, contact MasterQ32 (felix@ib-queissner.de)
to get access to the git repository (https://github.com/MasterQ32/TUST).
